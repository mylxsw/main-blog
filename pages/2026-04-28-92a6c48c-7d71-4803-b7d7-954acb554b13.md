# AI战局生变：微软OpenAI独家绑定终结，国产开源模型激战正酣

## 微软与OpenAI「七年CP」解绑：独家合作终结，OpenAI获准多云部署

微软与OpenAI于4月27日联合宣布完成合作协议的重大修订，标志着双方延续七年的深度绑定关系出现根本性转变。核心变化包括：

- **云合作独家限制解除**：OpenAI不再被限制只能与微软Azure合作，可向亚马逊AWS、Google Cloud等任何云服务商提供产品和服务。微软仍为首选云合作伙伴，但若Azure无法满足特定需求，OpenAI可自由选择其他平台。
- **知识产权非独家授权**：授权期限延续至2032年，但性质从“独家”变为“非独家”。
- **收入分成重新划定**：微软不再向OpenAI支付营收分成；OpenAI向微软的分成设总额上限，有效期至2030年，且与OpenAI技术进展脱钩。
- **AGI触发条款被取消**：原协议中当OpenAI实现AGI时微软分成权自动终止的条款已被移除。

OpenAI的GPT-5.5、Codex及托管智能体即将登陆Amazon Bedrock，企业客户可在AWS环境中直接调用OpenAI能力。亚马逊CEO安迪·贾西随即发文确认，AWS将在未来几周内通过Bedrock平台向客户提供OpenAI模型。

**深度解读**：这次“解绑”对双方而言是双向去依赖。微软早已启动“去OpenAI化”战略，内部训练Phi系列轻量模型、收购Inflection AI团队、接入Anthropic和DeepSeek等第三方模型。对OpenAI而言，除亚马逊外，公司已与Oracle、软银等建立新数据中心合作关系，2026年2月与亚马逊达成战略合作，承诺未来8年扩展至1000亿美元规模。此举打通了OpenAI重组与上市之路，也为全球AI市场的多极化竞争奠定基础。

### 马斯克起诉OpenAI开庭：法庭大战一触即发

同一天，马斯克起诉OpenAI案在加州奥克兰联邦法院进入庭审阶段。九人陪审团已完成遴选，开庭陈述定于4月29日开始。马斯克在X上攻击奥尔特曼为“骗子·奥尔特曼”，称其与格雷格·布罗克曼“偷走了一家慈善机构”。马斯克指控OpenAI违背了将AI实验室保持非营利组织的承诺，目前仅剩不当得利和违反慈善信托两项指控。OpenAI回应称“真相和法律都站在我们这一边”。责任认定阶段预计5月21日前结束。

### OpenAI布局硬件：正与联发科、高通合作开发智能手机

分析师郭明錤透露，OpenAI正与联发科、高通和立讯精密合作开发一款以AI代理为核心的智能手机。芯片由OpenAI与联发科、高通联合设计，立讯精密负责设计与制造。最大的创新点是手机不再依赖独立应用，而是用AI代理直接帮用户完成任务，通过本地小模型和云端大模型配合，根据任务复杂度灵活调用计算资源。产品规格和供应商名单预计2026年底或2027年Q1敲定，量产最早2028年开始。在此之前，OpenAI首款硬件产品预计2026年下半年发布，外界猜测可能是集成智能交互功能的耳塞。

## 开源模型军备竞赛升级：小米、DeepSeek同台角力

### 小米正式开源MiMo-V2.5系列，启动百万亿Token计划

小米4月28日宣布正式开源MiMo-V2.5系列模型，包含两款均支持100万上下文窗口的模型：面向复杂Agent和Coding应用的**MiMo-V2.5-Pro**，在GDPVal-AA和ClawEval榜单上位列全球开源模型第一；原生全模态模型**MiMo-V2.5**，支持文本、图像、视频和音频理解。采用MIT协议，允许自由商用、二次训练与微调。

同步启动的Orbit计划包括两部分：**百万亿Token创造者激励计划**面向全球用户免费发放100万亿（100T）Token权益，活动期4月28日至5月28日，申请通过者最高可获16亿Credits；**Agent生态共建计划**面向全球Agent框架团队提供Token限免支持。开源首日已完成阿里平头哥、亚马逊云科技、AMD、百度昆仑芯、燧原科技等多家芯片厂商的Day-0适配。

### DeepSeek-V4稳居国内第一，优惠活动延期至5月底

SuperClUE最新测评显示，DeepSeek-V4-Pro以70.98分位居中文大模型第一，Flash版本以68.82分紧随其后。V4系列采用混合专家架构（总参数1.6万亿，激活490亿），支持百万级超长上下文窗口。DeepSeek官方宣布Pro版本2.5折优惠活动延期至5月31日，当前输入价格（缓存命中）低至0.025元/百万tokens。

**深度解读**：小米与DeepSeek的同台角力，折射出国内开源模型竞争已从技术参数比拼转向生态构建。小米凭借Orbit计划吸引开发者、绑定芯片厂商构建护城河；DeepSeek则通过持续降价和开源策略巩固市场地位。这种竞争格局对开发者而言意味着更多选择和更低成本，但对中小模型厂商则构成严峻压力。

### DeepSeek核心骨干流失：近300人名单中10人已离职

据每日经济新闻报道，DeepSeek上周发布的V4技术报告出现了近300人的“研究与工程”作者名单，创始人梁文锋与所有研究员、工程师并列署名。名单中10人标注“已离职”，包括核心成员王炳宣（入职腾讯）、罗福莉（入职小米）、郭达雅（入职字节跳动Seed团队）、阮翀（加入元戎启行）等，覆盖基座模型、推理、OCR、多模态四条核心技术主线。2025年下半年至今，DeepSeek至少5名核心研发成员确认离职。

## 商业与产品动态

### GitHub Copilot6月起转向按量计费

微软GitHub宣布Copilot自6月1日起从固定额度订阅制改为按使用量计费模式。取消原有的“高级请求”（PRU）计数方式，改用AI Credits作为统一计量单位。用户每月订阅费用等值转换为Credits额度，用量按Token消耗（输入、输出、缓存）计算。基础代码补全和Next Edit建议功能仍免费，不消耗Credits。

订阅价格维持不变：Copilot Pro为10美元/月，Pro+为39美元/月，Business为19美元/人/月，Enterprise为39美元/人/月。企业用户可享受组织级共享额度池和6-8月额外促销额度（Business每月30美元，Enterprise每月70美元）。GitHub解释称，Copilot已从编辑器内辅助工具演变为能执行长时间、多步骤自主编码会话的智能体平台，原有计费方式难以为继。

### Adobe Firefly AI助手进入公测：跨应用智能体升级

Adobe于4月28日正式推出Firefly AI助手公测版，面向Creative Cloud Pro订阅用户及Firefly Pro/Premium付费方案持有者。核心升级在于从单一生成工具升级为跨应用智能体，用户只需用自然语言描述想要的效果，系统即可在Photoshop、Premiere、Illustrator、Lightroom等软件之间自动调度任务并交付成品。同步推出Creative Skills功能，包括批量照片编辑、人像精修、社交素材生成等预置工作流。Adobe强调用户始终保有完整控制权，可在AI执行过程中随时介入调整。

### 谷歌翻译20周年：推出发音练习功能

谷歌翻译迎来上线20周年纪念，目前支持约250种语言，月活跃用户超过10亿，每月翻译量约1万亿个单词。为庆祝里程碑，Android端推出“发音练习”功能，利用AI分析用户语音并提供实时反馈，已在美国和印度上线，支持英语、西班牙语和印地语。谷歌翻译从2006年的统计机器学习，到2016年转向神经网络，再到如今利用Gemini模型和最新TPU提升性能，已完成三轮技术迭代。

### 谷歌新版“每日简报”功能曝光

据APK拆解和行业消息，谷歌计划将Gemini中的主动式AI助手功能“你的一天”更名为“每日简报”。该功能通过整合用户搜索记录、电子邮件和Gemini聊天数据，自动推送日程、目标和重要事务通知，定位为Google Now在AI时代的升级版本。界面将显示“首要事务”和“当前目标”等板块。谷歌I/O 2026大会有望正式发布该功能。

## AI能力里程碑

### ChatGPT在东大和京大入学考试中超越人类顶尖考生

日本AI创业公司LifePrompt公布测试结果显示，搭载OpenAI最新模型的ChatGPT在东京大学和京都大学入学考试中表现远超人类考生。东京大学方面，理科总分比人类最高分考生高出50分，数学科目斩获满分。京都大学方面，法学院考试以771分刷新734分的人类纪录，医学院以1176分超越1098分的状元成绩。这与2024年AI无法通过东大入学考试的情况形成鲜明对比，标志着生成式AI在理解复杂学科知识和逻辑推演上已具备强大能力。

### 阿里达摩院实现全球首次肠癌“无感”检测

阿里达摩院联合广东省人民医院等机构研发出肠癌筛查AI模型DAMO COCA，从2.7万人的平扫CT影像中精准识别5例漏诊肠癌，敏感性达86.6%，特异性达99.8%（误诊率仅0.2%）。该模型在国际上首次提出“无感”肠癌机会性筛查方法——患者无需肠道准备，在常规体检的平扫CT中即可筛查肠癌。研究论文发表在欧洲肿瘤内科学会官方期刊《肿瘤学年鉴》（影响因子65.4）。这是达摩院“平扫CT+AI”多癌筛查路线继胰腺癌、胃癌之后的第三个突破。

### 中国科学院发布“磐石100”科研大模型体系

中国科学院正式发布“磐石100”大模型体系，覆盖数学、物理、材料、天文、环境、空天、地理、生命科学八大领域，标志着AI赋能科研从单点探索进入体系化创新阶段。底座“磐石·科学基础大模型”1.5pro版本基于650万条高质量科学推理数据训练，提供文献罗盘、创新评价和智能体工厂三大功能，其中智能体工厂已积累超过2000个科研工具。“数字细胞”模型30天内发现3个未知药物靶点并通过实验验证，“临空”大模型可进行复杂空天工程问题的深度推理。该体系已在50余家科研单位推广，覆盖百余个科研场景。
