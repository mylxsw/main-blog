# Anthropic 智能体技术全面升级，国产 AI 融资上市潮涌

## Anthropic 智能体技术体系全面进化

### [代码执行优化 MCP：智能体 Token 消耗锐减 98.7%](https://www.anthropic.com/engineering/code-execution-with-mcp)

Anthropic 提出通过代码执行模式重构 MCP（Model Context Protocol）框架，将工具集成从“全量加载”转变为“按需调用”。具体做法是将 MCP 服务器包装为代码 API，让智能体在独立的执行环境中处理数据，只将最终结果返回给模型。

**深度解读 & 洞察：**  
- **核心痛点解决**：传统方式下，每个工具的定义、中间计算结果都会挤占宝贵的上下文窗口（即 LLM 的“注意力预算”），导致成本飙升、性能下降。新方案将这一开销几乎归零。
- **效率飞跃**：实测显示，Token 消耗减少高达 98.7%，这直接意味着运行成本的大幅降低和复杂任务处理能力的提升。
- **能力扩展**：该模式天然支持更复杂的控制流（如循环、条件判断）、数据过滤、隐私保护（敏感数据不进模型）和技能状态的持久化积累。
- **权衡取舍**：此方案依赖于一个安全、可靠的代码执行沙箱环境，增加了系统架构的复杂性和潜在的安全风险，需要精心设计。

### [Agent Skills 框架发布：智能体可动态加载专业技能](https://www.anthropic.com/engineering/equipping-agents-for-the-real-world-with-agent-skills)

Anthropic 推出 Agent Skills 开放标准，允许开发者为 AI 智能体创建模块化的“技能包”。这些技能包包含特定领域的指令、脚本和资源，并采用“渐进式信息披露”机制，确保智能体只在需要时才加载相关内容。

**深度解读 & 洞察：**  
- **从通用到专业**：这解决了当前大模型“样样通、样样松”的问题，让 Claude 能像人类专家一样，在特定场景（如金融分析、代码审查）下展现出深度专业能力。
- **上下文效率**：通过按需加载，极大优化了上下文窗口的使用效率，避免了无关信息的干扰。
- **生态构建**：作为开放标准，此举旨在鼓励社区共建技能库，加速 AI 智能体的应用落地。

### [Claude Agent SDK 升级：从编码工具到通用智能体平台](https://www.anthropic.com/engineering/building-agents-with-the-claude-agent-sdk)

Anthropic 将原有的 Claude Code SDK 正式升级为 Claude Agent SDK，标志着其战略重心从“辅助编程”转向“构建通用人工智能代理”。新 SDK 围绕“收集上下文-采取行动-验证工作”的三步循环，提供了一整套开发工具。

**深度解读 & 洵察：**  
- **核心理念**：“给 Claude 一台计算机”，赋予其主动感知、操作和验证现实世界的能力，而不仅仅是被动回答问题。
- **功能完备**：支持多种上下文获取方式（文件搜索、子代理等）、多种行动手段（工具调用、代码生成、MCP 集成）和多种验证机制（规则、视觉反馈、LLM 裁判），为开发者提供了强大的灵活性。

### [上下文工程：AI 构建新范式](https://www.anthropic.com/engineering/effective-context-engineering-for-ai-agents)

Anthropic 提出“上下文工程”概念，认为在 LLM 时代，构建高效 AI 系统的关键已从传统的“提示词工程”转向对整个上下文资源的精细化管理。

**深度解读 & 洞察：**  
- **注意力即资源**：LLM 的上下文窗口是有限且昂贵的“注意力预算”，必须像管理财务预算一样精打细算。
- **方法论**：文章提出了三大策略应对长周期复杂任务：1) **压缩**（提炼关键信息）；2) **结构化笔记**（建立长期记忆）；3) **多代理架构**（分工协作）。
- **终极目标**：用最小的、高信号的 Token 集合，最大化期望的输出结果。

## 国内 AI 赛道资本与监管动态

### [月之暗面获 5 亿美元 C 轮融资](https://www.aibase.com/zh/news/24174)

月之暗面宣布完成 5 亿美元 C 轮融资，公司现金储备已突破 100 亿元人民币。CEO 杨植麟明确表示短期内不急于 IPO，并计划在 2026 年推行“双倍薪酬”激励政策，全力冲刺全球 AGI（通用人工智能）领先地位。

### [MiniMax 拟港股 IPO](https://www.qbitai.com/2025/12/366302.html)

MiniMax 已正式启动赴港 IPO 流程，计划募资超 6 亿美元，有望成为“港股 AI 第一股”。阿里巴巴和阿布扎比投资局已确认作为基石投资者领投，显示出其 B 端业务的强劲增长和毛利率的显著改善。

### [软银完成对 OpenAI 400 亿美元注资](https://www.aibase.com/zh/news/24140)

软银集团已正式完成对 OpenAI 的 400 亿美元巨额投资，持股比例超过 10%。这笔资金将主要用于支持 OpenAI 的“星门计划”（Stargate），即大规模算力基础设施建设，为其未来的模型训练和商业化铺平道路。

### [广电总局严打 AI 魔改视频](https://www.aibase.com/zh/news/24147)

国家广播电视总局宣布，自 2026 年起将开展专项治理行动，严厉打击利用 AI 技术对经典影视作品（特别是四大名著和经典动画）进行低俗化、恶意篡改的行为。此举旨在为 AI 内容创作划定清晰的法律和道德红线，保护文化资产。
