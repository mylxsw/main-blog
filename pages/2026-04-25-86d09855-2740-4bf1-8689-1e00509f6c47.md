# 谷歌400亿押注Anthropic，DeepSeek V4改写开源格局

## AI 资本博弈：谷歌 400 亿美元豪赌 Anthropic

### [谷歌拟向 Anthropic 投资至高 400 亿美元](https://www.ithome.com/0/943/379.htm)

谷歌先以现金投入 100 亿美元，对应 Anthropic 估值 **3500 亿美元**（约 2.4 万亿元）。若达到业绩目标，还将追加 300 亿美元。同时，Google Cloud 将在未来 5 年向 Anthropic 提供 5 吉瓦规模算力。

关键背景：
- Anthropic 最快可能在 **10 月进行 IPO**
- CEO 阿莫代伊曾在谷歌担任 AI 研究员，2021 年创立 Anthropic
- 谷歌此前已投入约 30 亿美元，并承诺提供最多 100 万块 TPU 芯片
- **Claude Code 爆红**是近期融资加速的主因之一
- 双方在 AI 编程市场存在直接竞争

**深度解读 & 洞察：**

- 这笔交易本质是「算力换忠诚」的资源绑定。Anthropic 最缺算力，谷歌最缺 AI 编程市场话语权。谷歌通过 TPU 和云服务将 Anthropic 锁定在自家生态内，同时获得大客户的稳定收入。
- 400 亿美元已超出普通财务投资范畴，更像战略防御——确保 Anthropic 不完全倒向竞争对手云平台。
- 但双方关系微妙：Claude 与 Gemini 在企业 AI 市场正面竞争，谷歌越依赖 Anthropic，就越难自己打赢这场仗。这种「既养对手又防对手」的格局在科技行业相当罕见。

**影响：**
- AI 编程工具赛道持续升温，Anthropic IPO 预期可能推高 AI 概念股估值
- 对英伟达构成间接利空——谷歌 TPU 作为替代方案获得更多部署机会

## 大模型前沿：DeepSeek V4 重写开源天花板

### [DeepSeek 发布 V4 系列模型及完整技术报告](https://www.qbitai.com/2026/04/406809.html)

历时 484 天开发，在架构、训练、后训练三层面大幅改动。推出两个型号：

- **V4-Pro**：1.6 万亿参数 / 490 亿激活参数，面向高级推理
- **V4-Flash**：2840 亿参数 / 130 亿激活参数，主打高效场景
- 均支持 **100 万 token 上下文**，MIT 开源协议

**核心架构创新：**
- **mHC（流形约束超连接）**：解决深层模型残差连接数值不稳定问题，通过 Sinkhorn-Knopp 迭代实现双随机矩阵约束
- **混合注意力机制（CSA + HCA 交替）**：CSA 负责 token 级精细检索，HCA 负责长距离全局信号汇总，是百万 token 效率的关键突破
- **Muon 优化器**替代 AdamW：基于矩阵正交化，在万亿参数规模的第二次大规模验证

**性能亮点：**

| 基准 | V4-Pro-Max | 对手 | 结果 |
|------|-----------|------|------|
| SimpleQA-Verified | 57.9 | K2.6: 36.9 | 领先约 20 分 |
| Codeforces Rating | 3206 | GPT-5.4: 3168 | 超越 |
| HLE | 37.7 | Gemini-3.1-Pro: 44.4 | 仍有差距 |

- **V4-Flash-Max** 仅激活 130 亿参数，推理任务打平 GPT-5.2 和 Gemini-3.0-Pro
- 自评落后最前沿闭源模型约 3-6 个月

**成本革命：** 单 token 推理 FLOPs 降至 V3.2 的 **27%**，KV cache 降至 **10%**。

**深度解读 & 洞察：**

- V4 的核心逻辑是「一直在删成本」——从 V2 的 MLA 开始，每一代都在压缩 KV cache、激活参数和计算量。百万 token 上下文不是新能力，而是把成本压到了可承担的水平。
- 混合注意力机制是报告最有分量的创新：两种机制各司其职，在长序列推理效率上取得突破。
- V4-Flash-Max 可能是最被低估的部分——仅 130 亿激活参数打平 GPT-5.2，按效率算可能是目前最极致的推理模型之一。
- DeepSeek 在万亿参数训练中公开承认「不知道为什么某些 trick 管用」，这种坦诚在大厂中相当少见。

**影响：**
- 开源模型竞争力大幅提升，可能压缩闭源模型的定价空间，利好 AI 应用开发者和企业用户
- 百万 token 上下文 + 极低成本，使长文档分析、跨仓库代码重构、长 agent 会话等场景的落地门槛显著降低

### [英伟达 Blackwell 平台适配 DeepSeek-V4](https://www.ithome.com/0/943/462.htm)

DeepSeek-V4-Pro 在 NVIDIA GB200 NVL72 上开箱性能超过 **150 tokens/sec/user**。开发者可通过 NVIDIA NIM 微服务部署，或使用 SGLang 与 vLLM 框架定制推理。

### [DeepSeek-V4-Pro API 限时 2.5 折优惠](https://www.ithome.com/0/943/528.htm)

优惠后价格：输入（缓存命中）0.25 元/百万 tokens、输入（缓存未命中）3 元/百万 tokens、输出 6 元/百万 tokens。活动至 5 月 5 日。DeepSeek 表示下半年昇腾 950 超节点上市后价格还将大幅下调。

### [Claude Code + DeepSeek V4-Pro 实测](https://www.oschina.net/news/437483)

开源中国实测结论：
- **切换成本极低**：修改一行 Base URL 即可，约一分钟完成
- **五大企业级场景表现扎实**：OA 审批、BI 大屏、报表、自动化部署、文档编辑均达标
- **两大限制**：不支持图片输入（最大硬伤）；超长上下文叠加复杂推理时不稳定
- **建议**：纯代码/文本任务的开发者可立即切换，重度依赖图片的用户等 Vision 模式开放后再迁移

## 芯片产业剧变：AI 推理让 CPU 重回舞台中央

### [英特尔库存 CPU 因 AI 推理需求售罄，股价暴涨创历史新高](https://www.ithome.com/0/943/410.htm)

今年 Q1，AI 服务企业对英特尔 CPU 需求异常强劲，连已减记的库存芯片都卖光了。

关键数据：
- 早盘一度大涨 **24%**，股价升至 83 美元，市值突破 **4160 亿美元**
- 突破 2000 年互联网泡沫时期高点
- 至少 **23 家券商**上调目标价，中位目标价从 46.50 美元升至 75 美元
- 今年以来股价上涨超 120%，去年已涨约 84%
- AMD 和 Arm 同步上涨超 11%
- 特斯拉已成为英特尔下一代 **14A 制程**客户
- CFO 坦承部分增长来自清库存和涨价，Q2 不一定能持续

**深度解读 & 洞察：**

- 过去几年 AI 训练所需 GPU（主要是英伟达）是绝对主角，CPU 被边缘化。但 AI 产业正从「训练阶段」大规模转向「推理阶段」（即模型训练好后开始大量应用），推理任务中 CPU 的性价比优势重新凸显。
- CEO 陈立武指出 CPU 与 GPU 配比正从 **1:8 迈向 1:1**，意味着每块 GPU 背后需要配套的 CPU 数量大增。
- 但 90 倍估值（远高于 AMD 的 37 倍和英伟达的 22 倍）已处于历史高位，短期风险不容忽视。

**影响：**
- 利好英特尔及传统 CPU 厂商，半导体板块整体受提振
- 对英伟达构成间接竞争压力——英伟达已罕见发布 CPU 产品应战
- 投资者需关注：AI 推理趋势是真实的产业变化，但 90 倍估值已透支部分预期，追高需谨慎

## 自动驾驶：燃油车智能化破冰，物理 AI 新范式

### [奥迪 Q5L 搭载华为乾崑智驾，全球首款城区领航辅助燃油 SUV](https://www.qbitai.com/2026/04/406960.html)

华为乾崑与一汽奥迪联合宣布，全新奥迪 Q5L 全系搭载华为乾崑智驾系统。

**燃油车智能化两大技术突破：**
- **感知架构**：定制车头双侧激光雷达方案（因保留奥迪家族化前脸，无法用车顶单激光雷达），试验标准达行业 2 倍
- **动力响应**：深度开发 VMM 车辆运动管理系统，解决燃油车动力链路比电动车更长导致的延迟问题

支持无高精地图的城区及高速领航辅助，包括无保护路口转向、遥控泊车、3 公里跨层记忆泊车、120 米循迹倒车等，并支持 OTA 升级。

**深度解读 & 洞察：**

- 燃油车占市场近六成，但此前高阶智驾几乎全是新能源车的专属标签。这次合作证明燃油车也能做，且不需要把整车重新设计。
- 华为以「技术供应商」身份切入 BBA 供应链，打破传统 Tier 1 角色定位——科技公司不再只是交付模块，而是深度参与产品定义，对汽车供应链分工模式是结构性变化。
- 中汽协数据显示新能源增速放缓（市占率 41.2% → 42%），燃油车仍有巨大存量。谁能先解决燃油车智能化，谁就能吃到这块蛋糕。

### [轻舟智航发布「轻舟乘风 MAX」：500 TOPS 跑世界模型](https://www.qbitai.com/2026/04/407026.html)

将世界模型部署到仅 500+ TOPS 算力车端，提出「物理 AI」定位。核心亮点：识别交警手势和潮汐车道等常识推理场景，130km/h 极限刹停且误触发率 50 万公里低于 1 次（行业平均 10-15 万公里一次）。首次明确 VLA（视觉-语言-动作）模型与世界模型是互补关系，非替代关系。

## AI 安全与治理

### [OpenAI 奥尔特曼就未通报枪击案嫌疑人正式道歉](https://www.ithome.com/0/943/456.htm)

OpenAI 的自动审核系统于 2025 年 6 月标记了加拿大枪击案嫌疑人（造成 8 人死亡）的 ChatGPT 账户，部分员工建议联系执法部门，但公司高层最终决定不通报。奥尔特曼就此正式道歉，表示公司已加强安全协议，今后类似情况将直接移交执法部门。

**深度解读 & 洞察：**

- 这起事件暴露了 AI 公司在「用户隐私保护」与「公共安全预警」之间的根本性矛盾。AI 聊天记录在多大程度上可作为安全预警信号？企业是否有义务主动报告？全球尚无明确法律框架。
- OpenAI 的默认选择——先封号但不通报——反映大多数科技公司的立场：保护用户隐私优先。但 8 条生命的代价让这种立场面临前所未有的审视。
