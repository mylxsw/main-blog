# Claude Opus 4.7发布、MCP爆严重漏洞、具身智能数据基建集中亮相

## 大模型巅峰对决：Claude Opus 4.7、Codex全面升级、Qwen3.6开源

### [Anthropic 发布 Claude Opus 4.7](https://www.ithome.com/0/940/099.htm)

Anthropic 今日正式发布 Claude Opus 4.7 模型，距上代仅两个月。新版本核心升级：

- **高级编码能力大幅提升**：用户反馈称此前需要人工严把关的最复杂编码工作，现在可放心交给 Opus 4.7
- **视觉与审美增强**：支持更高分辨率图像识别，生成界面、幻灯片和文档质量更出色
- **指令遵循与记忆改进**：更擅长基于文件系统的记忆，可在多会话长时间工作中保持关键信息
- **基准测试**：超越 Opus 4.6、GPT-5.4、Gemini 3.1 Pro，略逊于 Claude Mythos Preview

Anthropic 还同步推出 **AI 设计工具**，用户通过自然语言即可创建网站、演示文稿和落地页。消息公布后，Adobe、Figma、Wix 股价均下跌超 2%。
