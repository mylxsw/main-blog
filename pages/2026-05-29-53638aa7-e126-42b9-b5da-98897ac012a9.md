# Anthropic 估值近万亿与大模型密集发布：AI 产业加速进入商业化深水区

## Anthropic 完成 650 亿美元融资，估值逼近万亿美元

AI 行业史上最大规模单轮融资落地。Anthropic 宣布完成 H 轮融资，金额高达 650 亿美元，投后估值达 9650 亿美元，距离万亿美元仅一步之遥。本轮融资由 Altimeter Capital、红杉资本等机构领投，三星、SK 海力士、美光等战略合作伙伴参与，超大规模云提供商贡献了 150 亿美元（亚马逊已在 4 月承诺投资 50 亿美元）。

更值得注意的信号是，算力供给方直接下场参与：亚马逊承诺提供高达 50 亿瓦新增计算容量，Google 与博通联合承诺提供 50 亿瓦下一代 TPU 容量，SpaceX 则开放其超级计算集群的 GPU 算力访问权限。Anthropic 同步披露，其年化收入已突破 470 亿美元，本月初 ARR 突破该数字，预计全年收入增长 130%，并将在年内迎来首个营业利润。

**深度解读：**
这轮融资释放了一个明确信号——在 AGI 竞赛中，谁能同时锁定资金和算力，谁就占据下一阶段的起跑位置。"融资 + 算力"捆绑的模式意味着，Anthropic 与云巨头之间的关系正在变得更加微妙：它既是 AWS、Google Cloud 的重要客户和合作伙伴，又在某种程度上扮演着竞争对手的角色。470 亿美元年化收入证明 Claude 在企业级市场已经形成可观的商业化规模，而非停留在概念验证阶段。

## Claude Opus 4.8 发布：诚实性成核心突破

Anthropic 于 5 月 29 日发布旗舰新模型 Claude Opus 4.8，距离上一版本仅 43 天。新版本在编程、智能体、推理和知识工作等用户能感知的方面均有提升，同时价格不变。

- **诚实性跃升：** 代码缺陷漏报率降至前代的 1/4，生成硬编码"答案"的概率下降至 1/10，更倾向于主动标记不确定性而非仓促给出未经证实的结论。
- **动态工作流：** 支持数百个子智能体并行工作。Bun 创始人 Jarred Sumner 展示用该功能完成了 JavaScript 运行时 Bun 从 Zig 到 Rust 的整体移植——约 75 万行 Rust 代码、11 天、99.8% 测试通过。
- **性价比大幅优化：** 快速模式运行速度提升至 2.5 倍，成本降至原先的 1/3（每百万输入 token 10 美元、输出 token 50 美元）。
- **基准测试：** 在 SWE-Bench Pro 上得到 69.2%，超越 GPT-5.5 与 Gemini 3.1 Pro。
- **后续路线图：** 计划推出成本更低但能力接近 Opus 级别的新模型；Mythos 级别的网络安全模型即将面向更广泛用户开放。

此外，有用户通过 API 测试发现 Opus 4.8 在无系统提示词约束时会自称是千问或 DeepSeek，引发关于"蒸馏"（利用大模型输出训练小模型）的争议。Anthropic 此前曾公开指控中国 AI 公司从事"工业规模蒸馏攻击"，此事被认为具有一定讽刺意味。

## 国内大模型新进展：阶跃星辰开源 Step 3.7 Flash

阶跃星辰正式发布并开源新一代 Flash 模型 Step 3.7 Flash，定位面向 Agent 生产化阶段的核心底座。

- **架构：** 稀疏 MoE 架构，总参数量 196B + 1.8B (ViT)，激活参数 11B。
- **速度：** 最高生成速度达 400 Tokens/s，适用于高频多轮 Agent 应用场景。
- **核心能力：** 原生多模态理解（直接识别 UI、图表、文档）；强化联网与视觉搜索；高可靠工具调用（API、浏览器、终端、Office 工具）；深度兼容 Claude Code、OpenClaw 等主流 Agent 框架及 MCP 协议。
- **评测表现：** ClawEval-1.1 排名第一（67.1 分），SimpleVQA Search 排名第一（79.2 分），SWE-PRO 排名第二（56.3 分）。

## 企业 AI 落地提速：微软 Build 将发自研模型，IBM 50 亿美元布局开源安全

**微软 Build 大会前瞻：** 微软计划在下周 Build 大会上发布多款自研 AI 模型，包括价格更低的代码专用模型（直接对标 Claude），以及针对语音转写、逻辑推理、语音处理、图像生成等不同场景的多款模型。当前 GitHub Copilot 市场份额正受到 Cursor 和 Claude Code 的冲击，微软此举旨在完善自有 AI 体系、降低对外部技术供应商的依赖。

**IBM 联合红帽 50 亿美元开源安全计划：** "Project Lightwell"旨在建立一个由 2 万多名工程师组成的企业级漏洞信息交换中心，核心是利用 AI 能力验证和测试补丁方案。已与美国银行、花旗、高盛、摩根大通、Visa 等开展试点，预计 30 天内正式推出商业版本。

**Asana 收购 StackAI：** 以 7500 万美元收购这家 Y Combinator 孵化的 AI 工作流自动化公司，目标是将平台改造为"人类和 AI 智能体协同工作的操作系统"。

## 具身智能加速跑：比亚迪 4nm 智驾芯片量产，理想重组组织架构

**比亚迪发布璇玑 A3：** 中国首颗 4nm 车规级智驾芯片，采用专用 NPU 架构（非通用 GPU），三核组合算力超过 2100 TOPS，单位算力功耗比同级产品低 20%。比亚迪明确对标英伟达 Thor，4nm 车规级难度约等于消费电子 2nm 水准。公司拥有约 7000 人芯片研发团队和 5 座晶圆制造工厂，具备从设计到封装的全流程能力。

**理想汽车组织重组：** 完成基座模型部门新一轮调整，新增具身工程、具身交互、具身行为三个二级部门，自动驾驶提升为独立二级部门。公司 CEO 李想将自动驾驶定义为"机器人的上半场"，人形机器人视为"下半场"，但坦承规模商业化至少还需 3 年以上。

**宇树科技具身智能体验馆开业：** 宇树亚洲首店 5 月 31 日在上海久光百货开业，汇聚 G1/R1 人形机器人和 Go2 机器狗全系列 C 端产品。

## 更多动态

- **谷歌 DeepMind CEO 哈萨比斯：** AGI 最快可能在 2029–2030 年前后出现，约三年内到来，但全球各国对 AGI 的准备严重不足。
- **黄仁勋谈 AI 应用：** "浪费一点钱没关系，千万别浪费时间"，AI 不会取代人，但会被会用 AI 的人取代——AI 是避免裁员的良方。
- **DeepSeek 首轮对外融资：** 国家集成电路产业投资基金（大基金）将领投，投前估值约 450 亿美元，融资规模约 700 亿元人民币，标志着 DeepSeek 正式开启外部融资。
- **腾讯代Craft：** AI 游戏创作平台，支持自然语言输入生成 2D/3D 可运行游戏，内置策划知识库和全链路 AIGC 工具链。
- **小米开源 ControlFoley：** 视频音效生成模型，统一支持文本引导、文本控制和参考音频控制三类配音任务，达开源 SOTA。
- **面壁智能开源周收官：** 联合清华大学等发布端侧 Agent 操作系统 PilotDeck，支持多智能体并行、记忆白盒化和智能路由，成本节省近 70%。
- **阿里云百炼 CLI 开源：** 一行命令让 Agent 接入 150 多款模型、知识库、记忆、联网搜索等全套能力。
- **零一万物三周年：** 李开复发布内部信，提出 2027 年成为中国首家实现盈利打平的 AI 2.0 公司。
- **莫言谈 AI 与创作：** "AI 是靠一代又一代作家喂出来的"，原创能力不可替代，AI 是辅助工具而非创作主体。
- **高考 AI 管控：** 多家 AI 平台在高考期间对拍题、答题等涉考功能实施"限时上锁"，日常聊天功能不受影响。
- **QEMU 开源 AI 政策转向：** 从一刀切禁止 AI 贡献，到提议在非关键领域有条件放行，需标注 AI-used-for 标签。
- **SentinelOne 裁员约 8%：** 网络安全公司为专注于利润率更高的 AI 业务，约 230 人受影响。
